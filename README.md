# Yet another NHL Playoff Bracket Predictor

## Input Features
### Team Features
- Regular season record
- Goal differential
- Goals for
- Goals against
- PP %
- PK %

### Matchup Features
These features are tallied only when playing their opponent
- Matchup record
- Goals differential
- Goals for
- Goals against
- PP %
- PK %
- Faceoff %

### Playoff Features
- Running playoff series score

### Other Potential Features
